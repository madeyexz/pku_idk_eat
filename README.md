## 使用说明
1. 下载源代码
2. 打开命令行，输入`python3` 然后把源代码拖进去（当然直接用编译器跑也不是不行）
（到时候我可能会考虑把它变成一个`alias`这样以后就可以直接在terminal里面决定了😂）
